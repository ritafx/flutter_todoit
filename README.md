# TODO APP

Simple example of a Flutter Todo app.

### Data sources

Users - https://jsonplaceholder.typicode.com/users

Todos - https://jsonplaceholder.typicode.com/todos

### Features

- Login page
  - e-mail validation from API request
- Todo dashboard
  - Todo list
  - Update status
  - Add new todo
  - Delete todo
  - Logout
